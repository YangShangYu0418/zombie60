# Zombie60（暫名）— 喪屍末日經營 × 事件小說遊戲

目標：做出一款類似《60 Seconds!》節奏的「60 秒搜刮 + 日結算事件」生存經營遊戲，並加入小說式敘事與任務線；以「一位男主角 + 三位個性迥異的女角色」為故事核心。

技術建議（原型）：HTML + CSS + JavaScript（純前端），資料用 JSON 驅動事件與任務，存檔用 localStorage。

## 核心循環（Loop）
1. **搜刮階段（60 秒）**：移動/撿取/帶人進避難所。
2. **生存階段（每天）**：資源消耗 → 抽事件卡 → 做決策 → 狀態變化 → 推進任務線。

## 我們的共同工作方式
- 尚瑜：提供世界觀、角色喜好、事件靈感、文本風格取向。
- 悠雅：把故事拆成可玩系統（事件卡/任務線/狀態機），產出可執行原型，並試玩回饋平衡與敘事張力。
